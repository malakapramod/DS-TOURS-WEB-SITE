# DS Tours Website

This is a complete, mobile-responsive website for DS Tours, built with HTML, CSS (Tailwind), and JavaScript.

## How to Run
Simply double-click on `index.html` to open the website in your web browser. No installation or server is required.

## Features
- **Home Page**: Hero section, featured destinations, and introduction.
- **Destinations**: A gallery of top Sri Lankan tourist spots.
- **Services**: List of services offered (Transport, Hotels, etc.).
- **About Us**: Information about the company and mission.
- **Contact Us**: Contact form, Google Maps integration, and direct contact links.
- **WhatsApp Integration**: Floating WhatsApp button for quick contact.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Animations**: Smooth scrolling and fade-in effects.

## Customization
- **Images**: All images are currently fetched from Unsplash. You can replace them by changing the `src` attributes in the HTML files to your own local images.
- **Colors**: The color scheme is defined in the Tailwind config within the `<head>` of each file.
- **Contact**: Update the `mailto:` and `tel:` links in `contact.html` and the footer of each page with your actual details.

## Project Structure
- `index.html`: Main home page.
- `about.html`: About page.
- `destinations.html`: Destinations page.
- `services.html`: Services page.
- `contact.html`: Contact page.
- `js/script.js`: JavaScript for animations and mobile menu.
